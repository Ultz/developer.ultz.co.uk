Enum EffectSlotInteger

A list of valid AuxiliaryEffectSlot/GetAuxiliaryEffectSlot parameters.

Namespace: Silk.NET.OpenAL.Extensions.Creative
Assembly: Silk.NET.OpenAL.Extensions.Creative.dll
Syntax
public enum EffectSlotInteger : int

Fields

Name Description
AuxiliarySendAuto

This property is used to enable or disable automatic send adjustments based on the physical positions of the sources and the listener. This property should be enabled when an application wishes to use a reverb effect to simulate the environment surrounding a listener or a collection of Sources.
Effect

This property is used to attach an Effect object to the Auxiliary Effect Slot object. After the attachment, the Auxiliary Effect Slot object will contain the effect type and have the same effect parameters that were stored in the Effect object. Any Sources feeding the Auxiliary Effect Slot will immediate feed the new effect type and new effect parameters.
