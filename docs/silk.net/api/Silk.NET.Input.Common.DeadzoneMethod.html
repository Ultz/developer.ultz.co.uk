Enum DeadzoneMethod

Available methods to control the deadzone of a control stick.

Namespace: Silk.NET.Input.Common
Assembly: Silk.NET.Input.Common.dll
Syntax
public enum DeadzoneMethod : int

Fields

Name Description
AdaptiveGradient

A deadzone method where the reported value adapts to the range of the deadzone. If the value is within the deadzone, the reported value is 0. After exiting the deadzone, the reported value increases from 0 (when the actual value first exits the deadzone) to 1 (when the actual value is 1).
Traditional

The traditional deadzone method, where the reported value is directly proportional with the actual value until the actual value is within the deadzone - the reported value will be 0 in this case.
