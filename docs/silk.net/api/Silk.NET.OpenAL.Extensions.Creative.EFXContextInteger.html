Enum EFXContextInteger

Defines new integer properties on the OpenAL context.

Namespace: Silk.NET.OpenAL.Extensions.Creative
Assembly: Silk.NET.OpenAL.Extensions.Creative.dll
Syntax
public enum EFXContextInteger : int

Fields

Name Description
EFXMajorVersion

This property can be used by the application to retrieve the Major version number of the Effects Extension supported by this OpenAL implementation. As this is a Context property is should be retrieved using alcGetIntegerv.
EFXMinorVersion

This property can be used by the application to retrieve the Minor version number of the Effects Extension supported by this OpenAL implementation. As this is a Context property is should be retrieved using alcGetIntegerv.
MaxAuxiliarySends

Default: 2 This Context property can be passed to OpenAL during Context creation (alcCreateContext) to request a maximum number of Auxiliary Sends desired on each Source. It is not guaranteed that the desired number of sends will be available, so an application should query this property after creating the context using alcGetIntergerv.
